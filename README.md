# Розробка простої системи керування фінансами

## Finance Manager (C++)

Консольний застосунок, розроблений мовою C++, для обліку особистих доходів і витрат з можливістю бюджетування, аналізу витрат та захисту даних шифруванням.

---

## Технологічний стек

* **Мова програмування:** C++ (стандарт C++11)
* **Алгоритми:** Шифр Цезаря для захисту даних
* **Структури даних:** Вектори, мапи, файлові потоки
* **Платформа:** Linux / Windows

---

## Мета проєкту

Розробка простого та зручного застосунку для ведення особистого бюджету, який дозволяє користувачу контролювати фінанси, планувати витрати та аналізувати свої фінансові звички.

---

## Основні можливості

* додавання доходів та витрат з описом
* категоризація витрат (Їжа, Транспорт, Розваги, Інше)
* встановлення лімітів бюджету для кожної категорії
* автоматичні попередження при перевищенні бюджету
* детальні фінансові звіти з візуалізацією
* фільтрація транзакцій по датах
* експорт даних у формат CSV
* шифрування даних алгоритмом Цезаря
* збереження даних між сесіями

---

## Встановлення та запуск

### Компіляція

```bash
g++ main.cpp -o finance_manager -std=c++11
```

### Запуск

```bash
./finance_manager
```
